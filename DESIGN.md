## Next.js 구현 시 참고사항

### 컴포넌트 구조
```
src/
├── app/                    # App Router
│   ├── (public)/          # 공개 페이지 (상담 신청)
│   ├── (auth)/            # 인증 페이지 (로그인)
│   ├── (admin)/           # 관리자 페이지
│   └── (client)/          # 고객 페이지
├── components/
│   ├── ui/                # Shadcn UI 컴포넌트
│   ├── layout/            # 레이아웃 컴포넌트
│   ├── features/          # 기능별 컴포넌트
│   └── charts/            # 차트 컴포넌트
└── lib/
    └── api/               # API 클라이언트
```

### 라이브러리 추천
- **UI 프레임워크**: Shadcn UI (Tailwind 기반)
- **차트**: Recharts 또는 Chart.js
- **폼 관리**: React Hook Form + Zod
- **상태 관리**: Zustand 또는 React Query
- **날짜**: date-fns
- **아이콘**: Lucide React
