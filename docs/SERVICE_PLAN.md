# 준비스튜디오 서비스기획안

## 1. 서비스 개요

### 1.1 서비스 컨셉

> **"참고사이트만 주세요, 나머지는 저희가 다 합니다"**

기획서 작성 능력이 없는 소상공인/스타트업을 위한 웹개발 외주 서비스.
고객은 참고사이트 URL과 간단한 설명만 제공하면, 분석부터 화면설계까지 자동으로 처리.

### 1.2 핵심 차별점

| 기존 외주 | 준비스튜디오 |
|----------|-------------|
| 기획서/요구사항 직접 작성 | 참고사이트만 제출 |
| 견적까지 1~2주 | **30분 내 자동 분석** |
| 화면설계 별도 비용 | **샘플 화면설계 무료 제공** |
| 카톡/전화 소통 | 사이트 내 댓글 소통 |
| 불투명한 진행상황 | 실시간 진행률 대시보드 |

### 1.3 타겟 고객

1. **소상공인**: 쇼핑몰, 예약 사이트 필요
2. **스타트업**: MVP 빠르게 개발
3. **중소기업**: 사내 관리 시스템 필요
4. **개인**: 포트폴리오, 블로그

| 전달 | 고객 알림 | Resend (Email) + 푸시 |

## 6. 프로젝트 진행 프로세스

### 6.1 전체 흐름

```
[상담] ──► [계약] ──► [개발] ──► [검수] ──► [납품] ──► [A/S]

  │          │          │          │          │          │
  ▼          ▼          ▼          ▼          ▼          ▼
 자동       자동       수동       자동       자동       수동
```

### 6.2 단계별 상세

| 단계 | 설명 | 자동화 | 소요시간 |
|------|------|--------|----------|
| **상담** | 분석, 문서생성, 화면설계 | ✅ 완전 자동 | ~30분 |
| **계약** | 계약금 결제 | ✅ 자동 (PG) | 즉시 |
| **개발** | 실제 코딩 | ❌ 수동 | 2~6주 |
| **검수** | 피드백, 수정 | ✅ 반자동 | 1~2주 |
| **납품** | 산출물 전달, 잔금 | ✅ 자동 | 즉시 |
| **A/S** | 버그 수정 | ❌ 수동 | 3개월 |
